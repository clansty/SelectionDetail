## 安装方法

1. 游戏需要安装 [MelonLoader](https://github.com/LavaGang/MelonLoader)
2. 下载 [Release.zip](https://github.com/clansty/SelectionDetail/releases/download/1.0.0/Release.zip)，把里面的文件解压到游戏目录下
